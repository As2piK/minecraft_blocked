# minecraft_blocked

Datapack Minecraft Bloqué !
Basé sur Bordercraft avec mes propres settings :
- Les 5 premiers bloques en main ajoute 2 à la bordure
- Chaque nouveau bloc en main ajoute 0.1 bloque à la bordure
- Chaque achievement avec 2 bloques à la bordure

Pour démarrer :
- Créez une nouvelle map avec les commandes activées
- Faites /function bordercraft:activate

Pour pouvoir démarrer correctement, il faut un arbre dans la zone de départ. Le datapack ne le check pas, donc il faudra faire quelques resets
